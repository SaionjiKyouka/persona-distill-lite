# persona-distill-lite

A compact skill for turning a clearly named person or bounded topic into a **usable perspective skill**.

This project is intentionally small.

It keeps the parts that matter:

- first-hand-source preference
- minimal sufficient research
- 3 core mental models
- 5 decision heuristics
- expression DNA
- anti-patterns
- honest boundaries

It removes the parts that usually bloat cost and reduce reliability:

- large agent swarms
- excessive orchestration
- unnecessary intermediate files
- theatrical mimicry
- overgrown biographies

## What it is for

Use this skill to:

- distill a thinker into a perspective skill
- build a compact analysis advisor
- extract a judgment framework from a public figure
- update an existing persona skill with newer material
- create a bounded topic framework when a person is not the best unit

## Output shape

The generated skill is built around:

1. scope
2. operating rules
3. response workflow
4. 3 core mental models
5. 5 decision heuristics
6. expression DNA
7. anti-patterns
8. honest boundaries
9. key sources

## Design principle

Smaller and truer beats larger and vaguer.

The goal is not to create a dramatic clone.

The goal is to build a **usable reconstruction of judgment style**.

## Suggested invocations

- `/persona-distill-lite Richard Feynman`
- `/persona-distill-lite Nassim Taleb`
- `/persona-distill-lite product minimalism`

## Notes

Use first-hand material whenever possible. If the user provides books, transcripts, notes, or interviews, those should dominate the reconstruction.

When evidence is thin, the skill should shrink instead of pretending to know more.
